tabularx 宏包定义了一个新的环境 tabularx，其功能与标准的 tabular* 环境类似，但核心区别在于它通过调整特定列的宽度（而不是列间距）来使表格达到用户指定的总宽度。用户只需指定表格的总宽度，宏包会自动计算并调整标记为 X 的列的宽度，以填满该宽度。在表格 preamble（导言）中使用新令牌 X 来标记那些需要自动伸缩的列。这些 X 列最终会被转换为 p{<计算出的宽度>} 类型的列。该宏包依赖于 array 宏包。
可以使用 \tracingtabularx 声明或在加载宏包时使用 infoshow / debugshow 选项，来打印列宽计算的详细信息。默认情况下，X 列被定义为 p{...}（顶部对齐）。可以利用 array 宏包的 > 语法修改格式，例如 >{\small}X 或 >{\raggedright\arraybackslash}X（左对齐且不换行冲突）。可以通过 \newcolumntype 定义新的列类型（如 Y）来简化常用格式。可以通过重定义 \tabularxcolumn 宏来改变 X 列的基础类型（例如从 p 改为 m 以实现垂直居中）。默认所有 X 列宽度相同。可以通过调整 \hsize 参数让不同的 X 列拥有不同的相对宽度（例如一列是另一列的三倍宽），但需保持所有 \hsize 之和不变。如果普通列的总宽度已经超过指定总宽度，tabularx 不会将 X 列设为负值，而是发出警告 "X Columns too narrow (table too wide)"，并将 X 列宽度设为最小值（默认 1em），导致表格实际宽度超出设定值。从版本 2.12a 开始，支持生成带标签的 PDF（Tagged PDF），有助于无障碍访问。
