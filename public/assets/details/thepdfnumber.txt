该宏包的核心功能是将实数转换为符合 PDF 格式规范的最小化表示形式。在处理底层 PDF 内容（如颜色定义、透明度设置或计算生成的数值）时，通常需要确保数字格式严格符合 PDF 标准且长度最短。
去除冗余字符：移除前导零（例如 0045 → 45）。移除尾随零（例如 1.20 → 1.2）。移除不必要的正号 + 和多余负号（例如 +123 → 123, --123 → 123）。如果数字是整数，移除小数点（例如 1.0 → 1）。对于小于 1 的正数，移除整数部分的 0（例如 0.78 → .78），这是 PDF 格式允许的写法。
数值归一化（可选）：提供 \thepdfnumberNormZeroOne 命令，将数值限制在 0.0 到 1.0 之间（常用于 PDF 的透明度或颜色分量）。负数变为 0，大于 1 的数变为 1。
宏包提供了两个主要的可展开宏（只需两步展开即可得到结果）
\thepdfnumber{<number>}接受一个数字作为参数，输出其最小化表示。
+123 → 123
-01 → -1
1.0 → 1
0.0 → 0
0.78 → .78
+012.340 → 12.34
\thepdfnumberNormZeroOne{<number>}：将输入数字规范化到 [0, 1] 区间。
-456 → 0
0.010 → .01
1.001 → 1
4 → 1
