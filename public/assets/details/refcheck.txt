这是一个用于 LaTeX 2ε 的辅助工具包，主要作用是帮助作者检查和优化文档中的引用标签（labels）和参考文献（bibliography）。如果你定义了一个标签（如 \label{eq.1}），但在文中没有通过 \ref{eq.1} 使用它，Refcheck 会在编译日志（.log 文件）中发出警告。示例警告：Package refcheck Warning: Unused label ‘eq.2’ on input line 44.如果某个公式被编号但没有添加 \label，Refcheck 也会提示。示例警告：Package refcheck Warning: Unlabelled equation(3) on input line 46.如果你在 thebibliography 环境中定义了 \bibitem{book3}，但没有用 \cite{book3} 引用它，宏包会警告。示例警告：Package refcheck Warning: Unused bibitem ‘book3’ on input line 88.默认情况下（或通过选项 showrefs / 命令 \showrefnames），Refcheck 会在页面空白处显示所有标签的“键名”（如 eq.1, book2）。可通过 norefs 选项或 \norefnames 命令关闭此功能。类似地，参考文献键名可通过 showcites/\showcitenames 开启，nocites/\nocitenames 关闭。默认会将检查信息写入 .log 文件（选项 msgs）。可通过 nomsgs 选项或 \setoffmsgs 命令禁用日志输出。当使用 xr 包引用其他 .tex 文件的标签时，Refcheck 可通过 \refcheckxrdoc[prefix]{filename} 命令检查这些外部标签是否被正确使用。支持 AMS-LaTeX 和 hyperref 包（但需在加载 Refcheck 之前加载它们）。与 cite 包有较好的兼容性（自 v1.9 起）。对于隐式引用范围（如 \ref{a1}--\ref{an}），中间的 a2 到 a(n-1) 可能被误判为“未使用”。
