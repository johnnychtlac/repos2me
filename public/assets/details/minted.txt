minted 的核心作用是让你在 LaTeX 文档中优雅地排版编程代码，并利用 Python 的 Pygments 库对代码进行自动着色，使其更加美观和易读。minted 不像其他宏包那样在 LaTeX 内部实现语法规则，而是将代码传递给 Python 编写的 Pygments 库进行处理。这意味着它支持数百种编程语言和标记语言，且高亮规则非常准确。代码经过 Pygments 处理后，会被转换回 LaTeX 命令，因此你可以像使用普通 LaTeX 文本一样对代码进行排版（如设置字体、背景色、边框等）。新版 minted 使用了专门设计的 Python 可执行文件 latexminted，旨在满足受限 shell escape 的安全要求。宏包默认开启缓存功能，高亮后的代码会被保存，避免每次编译都重复调用 Python，从而显著加快后续的文档编译速度。你可以使用 Pygments 自带的各种样式（如 default, bw, colorful 等），也可以创建自己的样式。支持显示行号，甚至可以设置跨环境的连续行号（通过 lexerlinenos 选项）。支持自动换行，防止代码溢出页面。可以为代码块设置背景色（bgcolor），并自定义填充。提供 autogobble 选项自动去除代码的公共前导空白，方便从 IDE 复制代码。支持使用正则表达式（rangestartstring，rangestopstring 等选项）来只显示代码文件中的特定片段。提供了 listing 浮动环境，支持标题、标签和生成代码列表。需要安装 Python 3.8+ 以及 Pygments 库。通常需要启用 -shell-escape（或 MiKTeX 的 -enable-write18）参数来允许 LaTeX 调用外部程序。在 TeX Live 2024+ 中，由于 latexminted 被加入受信任程序列表，可能不再需要手动添加 shell-escape。
